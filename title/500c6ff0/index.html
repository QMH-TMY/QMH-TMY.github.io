浅谈现代操作系统历史

发表于 2018-10-13 | 分类于 操作系统 | 阅读次数:
| 字数统计: 2.3k | 阅读时长 ≈ 7

　

智能手机操作系统

智能手机的使用已十分普遍了。不论苹果或是安卓系统的智能手机都拥有巨大的市场量，拥虿甚广！可是，有谁能想到这个如此受欢迎的东西也就诞生十来年，也就是十年前汶川遭受灾害，国人感到悲伤的时候，安卓从山景市谷歌公司溜出来，并迅速引领了现代互联网的发展！

大概在2000年代，手机市场由诺基亚统治，当然，还有一些其他的手机，例如黑莓手机，奥巴马总统同贝爷外出打野时就用的黑莓定制无网络手机（为防止黑客窃听),与特总统天天推特比起来不知道高到哪里去了!
正是这个黑莓在那段时间开始搞手机系统，嵌入式. 与此同时，一名叫做Andy
Rubin的小伙子创立安卓，开始搞相机系统，为了处理照片。但是相机市场突然下滑，随即安卓公司开始转向于手机系统，并利用开源的Linux稳定内核作为其手机系统的核心，安卓在2005年被谷歌收购，经过加工处理，于2007年11月释出。并与同期发布的苹果手机系统同台角力！这就是我们今天看到的繁荣互联网世界背后的一点儿故事了。当然，苹果有另外一种手机操作系统，它的内核来源于BSD版本的Unix,同Linux属于同族不同分支的系统，所以严格来说，苹果系统是安卓系统的叔叔。他们的共同祖先是贝尔实验室的Unix系统，也是在这个私人实验室诞生了七个诺贝尔奖，卫星通讯，电影技术，激光技术等等均从这里诞生。当然，贝尔实验室的丹尼斯和里奇发明的操作系统和C语言一样的统治着这个星球的超级计算机，手机，电讯行业。（以后会专门写一个贝尔实验室系列文章，欢迎查阅)

PC操作系统

除了手机操作系统，我们常见的还有电脑上的操作系统，这种系统发迹于上世纪的美国，最开始分为多个派别，后来大体两派：一派是比尔盖茨的微软，一派就是各个大公司和大学，实验室合作的或独立开发的系统，其中贝尔实验室的Unix操作系统是最成功的。今天大家看到的Windows10操作系统源于1981年比尔盖茨花5万美元从Seattle Computer Systems（西雅图计算机系统公司)买的DOS(磁盘操作系统，就是一个黑框框)。聪明的比尔盖茨结合IBM的PC电脑，售卖该DOS系统，赚取大量利润，并以此为基础改进DOS系统，并最终演变出了Windows NT系统，也就是Windows95,Windows98,Windows2000。而后Windows NT不断进化，诞生了Windows XP， Windows Vista, Windows 7, Windows 8/8.1, Windows10 这就是地球上最好用最受欢迎的PC系统了，Windows系统是一个很好用，但是很难懂和很复杂的系统，内部为了兼容以往的软件和系统，保留了大量不合宜的代码，这就导致Windows是地球上架构最烂的系统，最容易受到黑客攻击的系统。客观讲，Windows是个很成功的产品，但不是像苹果那样的杰作。它有功于社会进步，但也有害于社会安全。除了一些特殊软件外，专业人员一般不用Windows.

另外一个电脑操作系统是苹果的电脑操作系统，源于上世纪乔布斯的励精图治，苹果电脑定义了现代操作系统的界面，下拉菜单，按钮，鼠标等配件。对，微软的电脑界面，菜单等等都是学的苹果的。

苹果的电脑操作系统同他的手机一样源于Unix变种BSD操作系统，是乔布斯被赶出苹果创立Nextstep公司后重新找人设计开发的达尔文操作系统的进化版本。该系统内核使用BSD，上层自主研发，终于在乔布斯回归苹果后得到了发展并壮大为今天的苹果操作系统,这系统充分利用了Unix的优点，使其特别流畅，其实安桌也来源于Unix，只是不同的子孙代而已，安卓一样的很流畅，应用多。

第三类电脑操作系统是Unix操作系统的变种Linux衍生版，因为Linux只是一个内核，通过结合不同的系统软件就是不同的款，但是核心一致。所以这一类操作系统特别多，因为开源，任何人可以拿过来改一下并发布。我现在写这个博客的操作系统属于Linux的Ubuntu18.04版本(我现在已经几乎不用Windows了)，是最成功的桌面版Linux操作系统，其他的Linux操作系统还有Debian,Centos,Redhat约上百款。安卓也是基于Linux的，只是是手机操作系统，可见,Linux是非常强大的,做手机和电脑端都吃得开。

服务器操作系统，超级计算机操作系统

由于这一类是属于行业的操作系统，民间几乎不可能看到，但是它却在背后默默的为亿万人民服务。
比如你购物需要进行交易结算，用任何的Ａpp
需要后台支持，这些都要靠服务器的操作系统来处理。目前服务器操作系统分为两大阵营，一派微软，一派Linux。微软在开发PC版本的操作系统的同时也开发了服务器操作系统，目前60%左右的服务器操作系统是Linux，因为它架构好，易修改，稳定。服务器必需要稳定，试想你在付款，一下卡了，你的钱不就白花了吗？这也是Linux的强项，Ｌinux的一款叫做Centos的系统据说创下过５年不关机不出事故的记录，对比微软动不动就出事故，犯卡，Ｌinux能统治服务器端自然就很好理解了。

另外一个很重要的操作系统是超级计算机的系统，因为超级计算机管理的存储和运算节点十分的庞大，所以系统必需稳定好用，这样，Linux再次占领该领域，全世界前500名的超级计算机几乎都是Linux系统的。

IoT物联网系统。

随着互联网的发展，人们提出基于万物互联的物联网技术，就是把所有的能连网的电器都用网连起来，比如空调，冰箱，汽车，卫星，手机，智能手表，智能自行车，智能椅子，甚至是床都可以用网连起来，这样就需要针对不同的东西，比如空调设计一种特殊的操作系统，因为空调内存有限，一种短小精干的操作系统才是最理想的。基于此，提出了物联网芯片Iot技术，通过中央服务器控制各种不同的物理器件，获取大数据，并存储在云服务器上，结合新的技术，例如人工智能，能为消费者提供更好的产品服务。所以，一款适合普通器件的操作系统必然是十分受欢迎的。目前，众多公司在利用Linux内核开发Iot技术，也就是嵌入式系统，如果把手机看做普通的物品，那么安卓系统也是一款嵌入式操作系统。这些都是可以看到的，尤其是一些智能产品和空调，冰箱，洗衣机里用的都是基于Linuｘ开发的嵌入式系统，也就是最初级的Iot技术,我认为，今后我们的操作系统还会细化，针对不同的应用场景，将会出现不同的新，他们都是IoT系统。于此同时，似乎也有必要推出一个应对物联网的统一接口协议，类似于POSIX的标准化协议，让各种物联网设备互联共享数据。未来的世界真是相当精彩啊!

总结

按目前的分类法，操作系统分为手机端，服务器，桌面端，物联网端，其中Windows在桌面和服务器端能做，苹果在手机和桌面端能用，Linux在手机，桌面，服务器，物联网iot上都能做！Linux是个非常有用的技术，希望普通人士也要学习Linux，这将会帮助我们更快更好的进入智能时代。未来，随着新技术的出现，应该还有新的系统出现，未来的世界将是万物互联的世界，一切都是数据，一切都可以计算，生活更便捷，通讯更发达，信息更畅通,未来可期!
